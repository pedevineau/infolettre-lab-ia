| | *Zoom sur un projet IA* \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         | |
| | *Le projet AIGLE*\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | *Philippe ALLAMAND,* Chef de Service Adjoint, Responsable de l'unité SIG à la Direction Départementale des Territoires et de la Mer de l'Hérault et chef de projet Aigle.\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | *En quoi consiste ce projet ?*\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   | |
| | Le projet consiste à détecter grâce à l'IA des constructions illégales (appelées cabanisation) sur des terres non constructibles, comme les zones naturelles et agricoles, les zones à fort enjeu environnemental, ou encore les zones à risques (inondation ou incendie). Nous utilisons pour cela les photos aériennes de l'IGN qui ont l'avantage de couvrir tout le département avec une résolution à 20 cm le pixel. l'IA va alors rechercher sur ces images les objets liés à la cabanisation : caravane, mobile-home, piscine, construction en dur, etc. Comme nous disposons de plusieurs millésimes de ces photos aériennes, qui sont mises à jour tous les 3 ans, nous pouvons comparer la présence de ces objets dans le temps et calculer automatiquement la prescription (qui est de 6 ans pour les constructions en dur). Nos contrôleurs essayent bien sur d'intervenir avant cette prescription.\\ | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | Nous envisageons à l'avenir d'utiliser des images satellitaires qui sont actualisées plus régulièrement mais qui ne couvrent pas tout le territoire.\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | L'objectif de Aigle est double : protéger d'une part les populations qui s'installent dans les zones à risques fort et protéger d'autre part notre environnement en luttant contre le phénomène de cabanisation qui est important dans le département de l'Hérault.\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | Aigle est complété par Lucca (pour LUtte Contre la Cabanisation). Il s'agit d'un outil d'assistance à la rédaction des PV et à leur suivi administratif. Il a été initié à la DDTM66 et est maintenant utilisé par plusieurs DDT.M en Occitanie ainsi que par des collectivités.\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | 54 communes sont embarquées dans cette lutte, et ont signé avec le préfet de l'Hérault une charte de lutte contre la cabanisation.\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | *Comment est t-il né ?*\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          | |
| | Nous avions postulé au premier AMI IA en 2018. A l'époque, il y avait déjà une forte volonté de notre directeur, Matthieu Gregory, pour renforcer nos outils de contrôle et développer des projets innovants. La vigilance territoriale reposait essentiellement sur une douzaine d'agents qui organisaient des brigades de contrôle. Ils utilisaient Google Earth pour faire du pré-repérage au bureau ou ne procédaient qu'à des contrôles terrain. Il nous fallait aller plus loin en précision, d'où le projet Aigle qui a permis de faire gagner du temps aux contrôleurs.\\                                                                                                                                                                                                                                                                                                                                  | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | En tant que lauréat de cet AMI, nous avons bénéficié d'un accompagnement de la DINUM et de la DITP pour monter un POC qui s'est avéré prometteur. Depuis, grâce à des financements internes et à d'autres AMI, nous avons pu faire évoluer Aigle, tant au niveau de ses performances que de son interface. Par exemple, il s'est enrichi de la possibilité d'émettre des fiches de signalement des objets suspects, qui sont transmises au maire, compétent en la matière. Le Gard envisage de l'utiliser.\\                                                                                                                                                                                                                                                                                                                                                                                                       | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | *Sur quelle technologie repose le projet Aigle ?*\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                | |
| | Dans le cadre du POC, nous avons dû créer une interface pour constituer une banque d'images des objets à rechercher. C'est ce qu'on appelle la phase d'annotation. Il a fallu trouver des images de référence plutôt dans un milieu naturel proche de ce que l'on peut trouver en cabanisation. Nous avons ainsi recensé environ 2000 caravanes et autant de navires, constructions en dur, etc. Plusieurs modèles Open source sont utilisés : YOLO pour le bâti et RETINA NET pour les autres objets.\\                                                                                                                                                                                                                                                                                                                                                                                                           | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | Deux critères nous permettent de calibrer le résultat de la détection (appelé inférence) pour définir un « seuil de confiance ». Ce seuil est à la croisée des chemins entre la sensibilité qui est la capacité à détecter tous les objets suspects et la précision qui est la part des vrais objets parmi ceux détectés. Nous avons accès à un curseur dans Aigle pour jouer avec ce seuil de confiance, de façon à voir plus de choses au risque d'avoir plus de faux positifs.\\                                                                                                                                                                                                                                                                                                                                                                                                                                | |
| | Ce seuil de confiance a quasiment doublé depuis le POC, nous sommes aujourd'hui entre 70 et 80 % de détection fiable, on a vraiment progressé.\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | *Un déploiement de ce projet à l'échelle nationale est-il envisageable ?*\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        | |
| | Au départ, la DDTM de l'Hérault seule n'aurait jamais pu développer un projet aussi technique. Cela demande des fonds importants. C'était très stimulant. C'est un projet ascendant qui vient du terrain. Il montre qu'on est plus sur des systèmes descendants, avec les services métiers qui doivent s'adapter aux outils nationaux. Désormais, la dynamique est inversée. Ce projet reflète l'importance pour un produit de répondre avant tout aux attentes des utilisateurs.\\                                                                                                                                                                                                                                                                                                                                                                                                                                | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | La difficulté c'est qu'une fois le POC terminé, on se retrouve un peu seul. Pour le déploiement du projet, j'ai contacté le Ministère de la Transition Ecologique qui suit attentivement notre projet, mais la question du déploiement reste encore en suspend. Il y a un vrai sujet là-dessus. Les projets locaux ont dû mal à se déployer s'ils ne sont pas sponsorisés dès le départ par un ministère. On a aussi parfois du mal à savoir ce qui se fait ailleurs pour valoriser et partager ce type d'initiatives. Je pense qu'il y a encore du travail à faire là-dessus. Il faudrait peut-être, à l'instar de l'accompagnement des POC, un accompagnement spécifique pour amener les projets plus loin.\\                                                                                                                                                                                                    | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | *Quels conseils donneriez-vous pour mener à bien ce type de projet ?*\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            | |
| | Il est essentiel de ne pas sous-estimer le temps passé sur ce type de projet. Pour ma part, les premiers mois de son lancement, le projet Aigle a nécessité un mi-temps. En plus de mes autres missions, j'avais des grosses journées. Il faut être prêt à s'engager.\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | Par ailleurs, quand je me suis lancé dans ce projet, je n'avais pas la connaissance des services métiers, notamment des contrôleurs. Je venais du monde de l'informatique pour rejoindre celui du SIG. Cette connaissance des services métiers est essentielle et nécessite de faire des points d'étapes réguliers avec les utilisateurs. Il ne faut pas non plus négliger la conduite du changement. Il est capital d'embarquer tout le monde et d'obtenir l'adhésion des services métier. C'est la prise en compte de leurs demandes qui fait avancer un projet.\\                                                                                                                                                                                                                                                                                                                                               | |
| | \\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | |
| | Enfin, l'analyse d'impact du projet est également importante. Nous avions été accompagnés dans le cadre de l'AMI sur ce point et nous avions déjà pu à l'époque identifier les points de vigilance, comme la montée en charge de notre service juridique liée à l'augmentation du nombre de PV et de soit-transmis\\                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               | |
